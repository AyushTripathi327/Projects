Algorithm Sorting Visualizer
A simple and interactive web-based tool built to visualize how classic sorting algorithms work under the hood. This project is created to practically understand Data Structures & Algorithms (DSA) and see the step-by-step execution of sorting logic in real-time.
Features
Real-Time Visualization: Watch the bars change colors and swap positions as the sorting happens.
Algorithms Included:Bubble Sort: Shows adjacent comparisons and swaps (O(n^2) time complexity).
Merge Sort: Visualizes the divide-and-conquer approach dynamically (O(n \log n) time complexity).
Complexity Display:** Shows the Time and Space Complexity of the running algorithm instantly on the screen.
Responsive Bars:** Automatically renders 10 random array elements with heights mapped directly to their values.
Tech Stack Used
Frontend: HTML5, CSS3
Logic & Animations: JavaScript 
Core JS Concepts:** Asynchronous programming (`Async/Await`, `Promises`), and DOM Manipulation.
