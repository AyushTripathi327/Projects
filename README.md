# Parking Management System

A console-based Parking Management System built in C++ to manage vehicle parking operations, parking slot allocation, fee calculation, and record management. The project uses Object-Oriented Programming (OOP), STL, and File Handling to provide a simple and efficient parking solution.

## Features

* Vehicle parking and slot allocation
* Duplicate vehicle validation
* Vehicle search using registration number
* Parking fee calculation based on duration
* Parking receipt generation
* Display all parked vehicles
* File-based data persistence
* Menu-driven interface

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* STL (Vector)
* File Handling

## Concepts Implemented

* Classes and Objects
* Encapsulation
* Vectors
* File Handling
* Search Operations
* Time-Based Fee Calculation

## How to Run

1. Compile the source file:

   ```bash
   g++ ParkingManagementSystem.cpp -o parking
   ```

2. Run the application:

   ```bash
   ./parking
   ```

# Sorting Visualizer

A web-based Sorting Visualizer built using HTML, CSS, and JavaScript to demonstrate how sorting algorithms work through real-time animations. The project helps users understand algorithm execution and complexity analysis visually.

## Features

* Bubble Sort Visualization
* Merge Sort Visualization
* Real-time sorting animation
* Dynamic array generation
* Time Complexity Display
* Space Complexity Display
* Interactive User Interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Implemented

* Data Structures and Algorithms
* DOM Manipulation
* Async/Await
* Promises
* Event Handling
* Algorithm Visualization

## Supported Algorithms

### Bubble Sort

* Time Complexity: O(n²)
* Space Complexity: O(1)

### Merge Sort

* Time Complexity: O(n log n)
* Space Complexity: O(n)

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Generate an array and select a sorting algorithm.
4. Watch the sorting process step by step.

## Learning Outcomes

* Understanding sorting algorithms visually
* JavaScript asynchronous programming
* DOM manipulation techniques
* Algorithm complexity analysis
